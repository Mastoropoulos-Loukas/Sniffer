μεταγλώττιση:
make all

εκτέλεση:
./sniffer [-p path]

Σχεδιαστικές επιλογές:
Για την πληροφορία των workers δημιουργήθηκε ουρά (δομή stl/queue)
Για την πληροφορία για την συχνότητα εμφάνισης των links δημιουργήθηκε map (δομή stl/map)
Τα αρχεία-αποτελέσματα αποθηκεύονται στον φάκελο ./res

Περιορισμοί:
Τα links διαχειρίζονται σωστά μόνο αν είναι της μορφής http://www.link
Δηλαδή είναι απαραίτητο το http:// και να ακολουθεί www.
